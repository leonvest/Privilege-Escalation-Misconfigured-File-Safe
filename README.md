# Privilege-Escalation-Misconfigured-File-Safe

Overview
This repository demonstrates and defends against a critical Linux privilege escalation vulnerability where world-writable /etc/passwd permissions allow attackers to gain root access without authentication. This is a real-world security issue that affects misconfigured Linux systems.
